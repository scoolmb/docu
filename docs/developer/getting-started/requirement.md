# 📦 Requirements

Before using Dropsy, make sure you have the following:

## **Wallet**

Any Solana wallet works:

- keypair
- Phantom
- Solflare
- BackPack
- Glow
- Ledger

## **Environment**

Recommended:

- Node.js ≥ 18
- Yarn or Npm ( package manager )
- Solana CLI (optional but helpful)

## **Tokens**

For creators:

- An SPL token mint
- Some SOL for transaction fees
- List of Eligible wallets

## **Knowledge**

Basic understanding of:

- Solana accounts
- Tokens ( standard / token-2022)
- Merkle trees (optional)

You’re ready to go.
