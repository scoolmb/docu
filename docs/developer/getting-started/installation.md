# ⚙️ Installation

Dropsy is built on top of **solana/kit**, which provides low-level Solana primitives and utilities.  
To ensure everything works smoothly, you must install **both** packages.

---

## 📦 Install Dropsy SDK

Using **NPM**:

```bash
npm install @dropsy/sdk
```

Using **YARN**:

```bash
yarn add @dropsy/sdk
```

## 🔧 Install solana/kit

Dropsy relies on core utilities exposed by solana/kit, such as Address helpers, PDA builders, serializers, and transaction tooling.
(Required Peer Dependency)

Using **NPM**:

```bash
npm install @solana/kit
```

Using **YARN**:

```bash
yarn add @solana/kit
```
