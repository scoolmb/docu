# 🚀 Version 1.0.0

Version **1.0.0** marks the first official release of the Dropsy SDK.  
This **MVP** launch focuses on the core, battle-tested instructions required to create and manage airdrops reliably at scale.  
By starting with a minimal but robust foundation, we ensure maximum stability for early builders while keeping the codebase clean and secure.

Going forward, the SDK will be expanded smoothly with new modules such as **delegation**, **vesting** schedules, and **affiliate** support — all introduced progressively to maintain compatibility and developer confidence.
